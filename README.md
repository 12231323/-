大学生创客项目实验
各位评委老师好，我是项目负责人尹强。我们的项目是“基于应力波检测的人体跌倒检测器”。一句话概括：我们让地板拥有了“听觉”，能在老人跌倒的瞬间，通过分析撞击产生的应力波来精准识别并报警，彻底解决了传统方案的隐私与佩戴痛点。

我们带来了三点核心创新：

第一，自研异构信息采集电路。 我们利用高精度应变片和RISC-V架构，将微小的地板形变转化为清晰电信号。这套方案相比传统传感器成本更低，且实现了信号采集与处理的硬件加速。

第二，自创时频域复合分析算法。 我们不止看震动大小，更看震动特征。通过时域分析冲击能量、频域检查低频占比，我们独创的“低过零率”指标能有效排除钥匙、书本掉落的刚性碰撞干扰，将检测准确率提升至95%。

第三，优化的Sim-to-Real迁移学习技术。 我们在Unity中构建数字孪生环境进行海量仿真，再将AI模型通过知识蒸馏压缩至极致，成功部署在成本极低的STM32F103单片机上。这让我们的处理器成本仅为传统视觉方案的十分之一。

市场前景方面，我国60岁以上老人超2.8亿，年均跌倒4000万人次。这是一个千亿级的智慧养老刚需市场。我们的产品无需穿戴、不拍视频，完美契合老人的尊严需求。目前项目已获两项国家级奖项和一项专利。

跌倒无小事，安全是大事。 我们希望用中国青年的硬核技术，守护每一位老人的安稳晚年。谢谢大家！
